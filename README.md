インタラクティブで美しい誕生日お祝いサイト。特別な瞬間やサプライズを演出します！

## ✨ 主な特徴

### 🕯️ インタラクティブケーキ
- 3段ケーキ、グラデーションデザイン
- 5本のろうそくを個別に吹き消せる
- 吹き消すと煙エフェクト
- 「ろうそくを吹く」ボタンで一括消灯
- 5秒後に自動で再点灯

### 🎊 サプライズエフェクト
- **紙吹雪**: カラフルな紙片が舞う
- **花火**: カラフルな花火アニメーション
- **風船**: 自然なアニメーションで浮かぶ
- **星空**: きらめく星の背景
- **パーティクル**: 下から上に舞い上がる

### 🎵 サウンド＆音楽
- 誕生日ソング自動再生（Web Audio API）
- ろうそくを吹くと効果音
- 祝福時のサウンド
- 音楽ON/OFFボタン

### 💝 自動メッセージ
- 5種類の誕生日メッセージ
- 3秒ごとに自動切替
- 全てのろうそくを消すと特別メッセージ
- アニメーション＆グラデーションテキスト

### 🎮 多彩なインタラクション
- **クリック**: ろうそくを個別に消す
- **キーボード**:
  - `Space/Enter`: 全部消す
  - `S`: サプライズ発動
  - `M`: 音楽ON/OFF
  - `1-5`: 各ろうそくを消す
- **モバイル**: マルチタッチでサプライズ

## 🎁 パーソナライズ機能

### 🔧 パーソナライズシステム
URLパラメータで名前・画像・テーマ色を切替

### 📋 管理パネル
`admin.html`で：
- 全ユーザー一覧
- 個別リンク生成
- コピーしてシェア
- 新規追加ガイド

### 🔗 URL利用方法
- **デフォルト**: https://nguyentrungnghia1802.github.io/Happy-Birthday/
- **個別**: https://nguyentrungnghia1802.github.io/Happy-Birthday/?person=john

### ➕ 新規ユーザー追加
1. `config.js`の`PERSON_CONFIGS`に追加
2. 画像を`res/img/`に追加
3. `PHOTO_SETS`にセット追加

### 🎨 テーマカラー例
- `#ffd700` - ゴールド
- `#4ecdc4` - エメラルド
- `#ff6b6b` - ピンクレッド
- `#96c93d` - グリーン

### 🚀 デプロイTips
- 画像は`res/img/`にアップロード
- ファイル名はconfigと一致
- `admin.html`でテスト
- 画像は1MB未満推奨

### 📱 シェア方法
1. `admin.html`を開く
2. コピーリンクをクリック
3. メッセ/メールで送信
4. 受信者は自分専用ページを閲覧

### メッセージ変更
`script.js`の`wishes`配列を編集

### エフェクト追加
`script.js`にfunction追加＆イベントで呼び出し

## 🐛 トラブルシューティング

### 音が出ない
- ブラウザの自動再生許可
- クリック後に音楽ON
- デバイス音量確認

### エフェクトが重い
- 紙吹雪数を減らす
- 不要なアニメ無効化
- デバイス性能確認

### モバイル問題
- viewport metaタグ確認
- 複数画面サイズでテスト
- click→touchイベント対応
